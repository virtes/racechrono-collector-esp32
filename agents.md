# Правила для агента

- Нельзя заливать прошивку на плату, если пользователь не попросил об этом явно.
- Для проверки сборки сразу запускать `make build` с повышенными правами
  (`sandbox_permissions: "require_escalated"`), потому что PlatformIO пишет
  lock/cache-файлы в `~/.platformio`, недоступный из sandbox. Не делать перед
  этим заведомо падающий обычный запуск и не повторять пользователю длинное
  объяснение про `~/.platformio`.
